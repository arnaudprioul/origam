## What changed

origam 2.5.1 is a pure housekeeping patch — no new components, no API changes, no breaking moves. The version bump exists to signal a deliberate structural shift that brings the repo in line with a clean DS-module shape.

## Spec files moved to `tests/TU`

All 48 `.spec.ts` files that lived alongside their tested source in `src/` have been relocated to `tests/TU/<same-tree>/<file>.spec.ts`. The `TU` directory is where unit tests were always intended to live; they just never got there during the initial velocity phase.

Every import inside each spec was rewritten from a relative path to the `@origam/…` alias so the files resolve correctly regardless of their depth in the new tree. The two stale `__tests__/` subdirectories that were dragged along during the move were flattened — no more `tests/TU/…/__tests__/…` double-nesting.

`vitest.config.ts` `include` is now narrowed to `['tests/TU/**/*.spec.ts']`. The `tests/e2e/**` and `tests/a11y/**` trees remain excluded from the unit runner — they run separately via Playwright.

All 753 specs pass green from the new location.

## Dev playground removed

`src/App.vue`, `src/main.ts`, and the root `index.html` were a minimal Vite dev playground left over from the earliest prototyping days. Since v2.x, Histoire is the primary dev path — the playground was vestigial. Dropping it cleans roughly 500 B from the published tarball and around 50 lines of source that would never be maintained.

`vite.config.ts` (the dev-server config, not the lib build) was deleted alongside. The lib build uses unbuild via `build.config.ts` and is untouched.

## Package scripts cleaned up

- `server:dev` and `server:preview` removed (no playground to serve).
- `server:build` renamed to `lib:build` — it builds the npm module, not a "server".
- `dev` chain simplified: `story:dev && docs:dev`.
- `preview` chain simplified: `story:preview && docs:preview`.
- `prepublishOnly` updated to call `lib:build`.

## Tarball impact

Before: 3 177 files (including stale `dist/src/App.vue`, `main.ts`). After: 3 173 files — roughly 5 kB lighter, source-only.

## Upgrading

No code changes needed. The package surface is identical to 2.5.0.

```bash
npm install origam@2.5.1
```
