# Introduction

**Origam UI** est une bibliothèque de composants Vue 3 native, conçue pour offrir une expérience de développement cohérente et rapide au sein de l'écosystème Origam.

## Autonomie
Contrairement aux versions précédentes, cette bibliothèque est désormais **indépendante** et ne nécessite pas Vuetify pour fonctionner. Elle utilise ses propres styles et logiques de composants.
